Что такое Marshall AI Listener?

Пассивная AI-система мониторинга диспетчерских Telegram-чатов логистической компании Marshall (500+ партнёров).

Система слушает чаты, извлекает структурированные данные через LLM, детектирует критические события по бизнес-правилам и отображает алерты в реальном времени на веб-дашборде для менеджеров.

Проблема: хаос в 4+ мессенджерах, потерянные документы, отсутствие контроля статусов, штрафы до 100% стоимости рейса.

Решение: AI-слушатель, который автоматически парсит сообщения и выделяет критические ситуации.

Спецификации модулей (Спринт 3)

Спринт 3

Chat Listener

Telethon MTProto, подключение к 3 чатам + DM, event listener, raw_messages, reconnect, rate limiting.

 Спринт 3

AI Parser

LLM-парсинг через MiniMax M2.5 + Groq fallback, извлечение 9 полей, confidence scoring, дедупликация.

 Спринт 3

Alert Engine

5 типов алертов, 3 уровня severity, правила для 6 заказчиков (YAML), расчёт штрафов, hot-reload.

 Спринт 3

Схема данных

PostgreSQL 15: 5 таблиц, индексы, CHECK constraints, 3 view, seed data для 3 тестовых чатов.

API и дизайн (Спринты 4-5)

Спринт 4

Dashboard REST API

FastAPI + asyncpg: 9 endpoint-групп, JWT auth (3 роли), пагинация, фильтрация, Pydantic v2 schemas.

 Спринт 5

Дизайн дашборда

UX-анализ, JTBD экранов, ASCII-вайрфреймы 5 экранов, компонентная модель, REQ/AC требования.
