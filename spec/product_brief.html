Продуктовый бриф

Marshall AI Listener — Sprint 0 MVP

Пассивная AI-система, которая слушает диспетчерские чаты, структурирует данные через LLM и показывает менеджерам критические события в реальном времени — для снижения штрафов и хаоса в логистике.

Jobs-to-be-Done

Диспетчер (косвенный)
Когда я координирую рейсы через 4+ мессенджера одновременно,
я хочу продолжать работать как обычно,
чтобы система сама помогала менеджеру видеть проблемы.
Менеджер (прямой)
Когда случается проблема с рейсом (опоздание, поломка, простой),
я хочу увидеть её в дашборде за секунды,
чтобы оперативно связаться с диспетчером и избежать штрафа.
Руководство Marshall
Когда заканчивается рабочий день,
я хочу посмотреть аналитику: рейсы, проблемы, штрафы,
чтобы оценить операционную эффективность.

MVP-скоуп Sprint 0

Sprint 0 — БЕСПЛАТНЫЙ пилот. Цель: показать ценность, зацепить клиента для платных спринтов ($250/час).

S0-F01

Chat Listener

Подключение к 3 Telegram-чатам через Telethon (MTProto). Чтение всех текстовых сообщений + DM в реальном времени.

Входит:

  • Telethon MTProto как pluggable-адаптер
  • 3 групповых чата + DM
  • Asyncio event listener
  • Graceful shutdown + JSON logging
  • Rate limiting 30 сообщений/сек

Не входит:

  • Bot API (Sprint 1)
  • Голос/STT (Sprint 1)
  • Max, WhatsApp, Viber
S0-F02

AI Parser

LLM-парсинг сообщений: извлечение trip_id, маршрута, статуса, заказчика, urgency. MiniMax M2.5 + Groq fallback.

Входит:

  • 5-слойный промпт-пайплайн
  • 9 извлекаемых полей + confidence
  • Fallback: MiniMax → Groq
  • Дедупликация (5 мин окно)
  • confidence=0 при ошибке парсинга

Не входит:

  • STT (голос в текст)
  • Фото/документы
S0-F03

Alert Engine

Детекция критических событий на основе правил 6 заказчиков. 5 типов алертов, 3 уровня severity. Только дашборд.

Входит:

  • Правила 6 заказчиков (Тандер, WB, X5...)
  • 5 типов: delay, failure, downtime, safety, docs
  • HIGH / MEDIUM / LOW severity
  • Reviewed / Resolved статусы

Не входит:

  • Отправка в Telegram (Sprint 1)
  • Автоэскалация (Sprint 1)
  • SMS / push
S0-F04

Dashboard

Одностраничный веб-дашборд: активные рейсы (таблица) + алерты (лента). FastAPI + Chart.js, обновление каждые 3 сек.

Входит:

  • REST API: /trips, /alerts, /stats, /chats
  • Базовая авторизация (пароль в .env)
  • 3 графика: рейсы, время, алерты
  • Фильтры: заказчик, дата, статус

Не входит:

  • Мобильная версия
  • Экспорт Excel
  • OAuth/SSO
